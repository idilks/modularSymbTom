Hypotheses:

(I shorten belief formation patch as bf going forward)

  H1: abstract[e] ≠ control (belief tracking is separable from location retrieval)

 T[f]he control condition has the exact same narrative structure but only swaps which location token is which. If they overlap heavily, what we called "belief tracking" is just location token processing with extra steps.

  - metric: jaccard similarity of significant head sets, overlap count[g]

  - this is the most important hypothesis. if it fails, the paper's framing needs rethinking.

We find: it's strongly supported! spearman rho = -0.03 and -0.05 (bf and end). jaccard 0.00 and       0.02. abstract and control find almost completely different heads.

  H2: abstract ≈ photo[h] (the model does generic state tracking, not mentalizing)

  the photo condition replaces the agent with a photograph, with the same logical structure (something records state at T1, world changes at T2, question asks about the record) but no mentalizing required.

 if abstract and photo find the same heads:

  → the model tracks "deviation from a recorded state" generically. it doesn't have theory of mind; it has state-change tracking.

  if they find different heads:

  → there's something specific to agent-based reasoning. actual mentalization circuitry.

We find: the shared head L35H30[i] appears in both abstract_end and photo cross-template sets. Based on the pictures this seems a bit low and I will investigate how this looks like per template.

  H3: belief_formation patch is more specific than end patch (belief is computed locally) OR they have a big overlap

  - prediction: abstract × belief_formation finds fewer, more targeted heads than abstract × end

  - if confirmed: belief representation is computed at the epistemic access point and just propagated forward    

  - if rejected: belief tracking is distributed — the model re-computes or integrates across the full narrative
